## 一、感測器監控程式(Monitor_Software_Demo.py)： ##
    （1）設有帳號密碼權限管制功能。
    （2）可顯示感測器即時狀態。
    （3）可同時連線多台感測器。
    （4）收集的資料可轉換為報表及圖表。
    （5）可輸出報表（*.csv）及圖表（*.png）。
    （6）有異常警報功能（可開關）。
